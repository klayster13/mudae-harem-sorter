# mudae-harem-sorter
Web tool to sort and manage Mudae harem exports with drag-and-drop, locking, and $sm command generation. Supports notes and custom images.

**Live version:** [Open Sorter](https://yourusername.github.io/mudae-harem-sorter)

## Changelog

### v1.0 (April 2026)
- Initial release
- Monochrome dark UI with #57bb8a accents
- Robust import parser supporting notes (`| sX`)
- Drag & drop reordering with locked section
- Per-card controls (↑ ↓ SERIES LOCK)
- Search by character or series
- Sort by name/series/kakera + shuffle unlocked
- Multi-box $sm / $smpos command generation
- Navigation sidebars + Return to Last Position
- Full help modal with instructions
