Mudae Harem Sorter - Instructions

1. Save System

• Auto Save: Every change (drag, lock, sort, import) is automatically saved to your browser.
• Save: Click to choose a filename and download a backup JSON file.
• Load: Restore a previous save (choose file or paste JSON).

2. Getting Your Export from Mudae

In Mudae, type one of these commands to generate a clean export thread:

  • $mmak-i-s → Use this for base kakera values
  • $mmaki-s → Use this if you want key kakera values

Copy the entire output and paste it into the Import box.

3. Importing

Click 📥 Import, paste the full export, and click Import.

4. Search

Type a character name or series name in the top bar and press Enter.
• Character search jumps to that character.
• Series search jumps to the highest character from that series in your current list.

5. Sorting

Use the buttons on the right sidebar:

  • Sort by Name – First click: A-Z • Second click: Z-A
  • Sort by Series – First click: A-Z • Second click: Z-A
  • Sort by Kakera – First click: High to Low • Second click: Low to High
  • Shuffle Unlocked – Randomly shuffles only the unlocked characters (locked ones stay at the top).

Clicking a different sort button resets to its normal order.

6. Per-Card Controls

  • ↑ – Move to bottom of locked section
  • ↓ – Move to very bottom
  • SERIES – Move all unlocked characters from same series below locked section
  • 🔒 Lock – Lock this character + everything above it
  • 🔓 Unlock – Unlock this character + everything below it

7. Navigation Sidebar

Left sidebar buttons let you quickly jump around:

  • ↑ Go to Top, 🔒 End of Locked, ↓ Go to Bottom
  • ↩ Return to Last Position – Jumps back to the exact scroll position you were at before using any other navigation button.
  • 📜 Jump to Output

8. Other Features

• Drag cards to reorder freely (locked cards stay pinned at top).
• 🔓 Unlock All removes all locks.

9. Generate Commands

Click 📋 $sm to create ready-to-paste Discord commands (automatically split and chained with $smpos).
