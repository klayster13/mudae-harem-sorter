Mudae Harem Sorter v1.5.0 - Help

1. Getting Started

In Mudae, use one of these commands to generate your export:

  • $mmai-k-z+z!s — Base kakera values
  • $mmai-kz+z!s — Key kakera values

Important: Copy everything starting from the first series header down to the bottom. Do not include the total kakera line or harem name at the very top.

Smart Merge: Updates kakera, spheres, perks, series, and notes. Prompts for image changes if needed.

2. The Grid & Controls

  • Drag & Drop — Reorder freely. Locked cards stay pinned at the top.
  • 📸 Camera Icon — Hover image and click top-left camera to update picture.
  • Ctrl + Click any card to copy character name.
  • Bottom buttons: ▲ ▼ SERIES Lock/Unlock

3. Locking & Navigation

Locking System: Locked characters are protected at the top. Locking one character also locks everything above it.

  • ↑ Go to Top
  • 🔒 End of Locked — Jump to the last locked character
  • ↓ Go to Bottom — Jump to the end of your harem
  • ↩ Return to Last Position
  • 📜 Jump to Output — Scroll to the generated $sm commands

4. Sorting Options (Right Sidebar)

  • Sort by Name — A→Z (first click), Z→A (second click)
  • Sort by Series — Groups by series, then high→low kakera
  • Sort by Kakera — High→Low (first click), Low→High (second click)
  • Sort by Spheres — High→Low (first click), Low→High (second click). Ties broken by kakera high→low, then name A→Z.
  • Sort by Note — Moves all noted characters to the top (grouped alphabetically by note, then high→low kakera within each note). Characters without notes go to the bottom (sorted alphabetically by name).
  • Shuffle Unlocked — Randomly shuffles only unlocked characters
  • Unlock All — Removes every lock

5. Notes & Image Updates

  • Notes are preserved during smart merge and updated if changed in the new export.
  • Use the 📸 camera icon on any character image to manually update the picture.

6. Search

The top search bar finds matches in character names, series names, notes, or SP values.

7. Generating $sm Commands

Click 📋 $sm to create ready-to-paste Discord sorting commands. The output is automatically split into multiple boxes with proper $smpos chaining when needed. Use the copy button on each box.

8. Harem Statistics

Click 📊 Stats to view comprehensive harem statistics including total kakera, perk distribution, series breakdown, and your most valuable characters.

Thank you for using Mudae Harem Sorter!
